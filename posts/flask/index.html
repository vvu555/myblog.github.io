Mac系统下使用VSCode搭建flask环境

flask Mac OS VScode

准备工作

系统版本：macOS Catalina 10.15.7

软件：vscode 1.74.3

python版本：3.8 （低于3. X的小伙伴一定要先更新python版本，mac系统设置默认python版本请看问题2）

开始

第一步：打开在mac电脑上新建一个文件夹（你觉得合适的任意位置），以我的为例“新建Flask文件夹“

第二步：打开vscode，再点击打开文件夹选择我们上一步创建好的文件夹右下角的打开

第三步：勾选信任并选择“是”

第四步：在vscode界面点击左上角的终端，选择新建终端

第五步：在终端输入以下命令并回车：

python -m venv env

这里要注意的是你的终端所在目录时在你创建的Flask文件夹中（红色框框右上角的那个目录嘻嘻）

3、上面这个步骤出现的问题问题及解决方法

如果你和我一样这里出现了如下情况，出现了 “No module named venv” 的错误，不用疑惑这个问题通常是因为你的 Python 安装没有包含 venv 模块，或者该模块未被正确安装。

原因：在 Python 3.3 及以上版本中，venv 是一个内置模块，你应该可以直接使用它。但在一些 Python 3.x 版本中，venv 模块可能需要单独安装。在这种情况下，你可以使用以下命令来安装它：

python -m pip install virtualenv

也许还会出现以下情况

请查看电脑中的python版本是否过于老旧（查看python版本代码python --version），通过跟新到最新的python版本，以下是更新python的指令

使用以下命令安装最新版的Python：

brew install python

如果你想安装特定版本的Python，可以使用以下命令：

brew install python@x.x   #将"x.x"替换为特定版本号，如2.7或3.9

等待安装完成，然后你可以再次尝试运行 python -m venv env 命令来创建一个新的虚拟环境。 如果你仍然遇到问题，请继续看问题2

问题2

原因：在 Mac 系统中，默认会安装一个 Python 版本，这个版本通常是操作系统所依赖的版本，并且它不会因为你在系统中安装了其他版本的 Python 而被替换掉。当你在 Mac 系统下更新 Python 版本时，你需要执行一些额外的步骤才能确保更新生效。

以下是更改mac系统下默认 Python 版本的步骤：

检查Python3的文件位置指令

which python3

随后输入打开配置环境变量文件指令添加环境变量

open ~/.bash_profile

加入设置指令

alias python='这里复制你刚才查询到的python3的地址'

然后按住command+s保存后直接关闭文件，随后再输入指令以达到环境变量立即生效的目的

source ~/.bash_profile

然后我们再在控制台中输入指令查看python版本

python --version

如果你仍然遇到问题，请确保你已经正确地执行了以上步骤。 解决完这些问题后回到vscode，这里注意要把之前的终端关闭重新新建一个终端，为了保险起见查询一下当前python版本，随后再次输入步骤五的指令,可以看到成功创建

python -m venv env

第六步：在右上角选择查看—指令面板，也可以使用快捷键shift+command+p进行快速选择。

这里输入ptthon又出现问题了，别急下面有解决办法

问题3

如果出现“没有匹配的命令”是因为没有安装python插件

点击左上的扩展，也可以按住shift+command+x进行快速选择。随后搜索python，选择箭头所指的进行安装

再次输入第六步的指令就有了！图里的快捷键时唤出终端啦就别再按一次了

Python: Select Interpreter

第七步：回车，并选择带有“env”的Python解析器

这时候我们再新建终端窗口是会发现此时终端目录前带有（env）

第八步：在新建的终端窗口输入指令，等待安装完成

pip install flask

走完进度条会黄色提示需要更新，这里只需要复制提示语句里的pip install --upgrade pip到下面回车运行即可

第九步：安装flask，输入指令pip install flask

注意：如果这里还会出现安装不了的情况可能是你的pip插件安装在了之前版本的python目录下了，这时候不要慌只需要知道你的pip安装位置，并把它的位置加到你电脑里的环境变量中就可以了，代码如下：

export PATH=你的插件安装位置/bin:$PATH

安装完成后在FLASK目录下创建app.py文件（直接后缀也是直接输入），在新建的py文件中写入如下代码：

from flask import Flask

app = Flask(__name__)

@app.route('/')
def hello_world():
    return 'Hello, World!'

这里记住要command+s保存才行，要不然运行下面的命令会提示找不到！

第十步：在终端输入指令python -m flask run最后在出来的提示中的端口，鼠标放上去按住command点击进入浏览器中即可 image-20230309021454691.png
