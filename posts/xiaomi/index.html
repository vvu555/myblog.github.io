安卓 2023年3月12日

Mac系统下通过adb线刷安卓系统

安卓 Mac OS 线刷 adb

准备工作

系统版本：macOS Catalina 10.15.7

手机型号：小米8屏下指纹版

环境安装：brew、android-platform-tools

操作

第一步：查看自己电脑中有没有安装brew，在控制台输入“brew”后会出现如下

如果没有可以使用此命令来安装，并且能自动配置国内镜像资源

/bin/zsh -c "$(curl -fsSL https://gitee.com/cunkai/HomebrewCN/raw/master/Homebrew.sh)"

按照指示一步一步安装即可

遇到的坑：在个别方法中会出现连接不上或者下载之后无法安装的情况例如

brew install android-platform-tools
brew cask install android-platform-tools

如果使用了这些都不能成功安装那就可能是网络的的问题

第二步：别急着关闭，接着输入安装android-platform-tools指令

brew cask install android-platform-tools

但是这时还是会提示错误，那是因为书写不对，要改成

brew install android-platform-tools --cask

等下在安装就可以，安装完后验证安装是否成功输入“adb”，显示如下就代表安装成功了

这时最重要的就做完了，下面进入刷入安装包，在这之前我们要根据自己手机型号下载不同的rom包，每个系统版本的rom包都可以在这个网站获取：https://xiaomirom.com/，这里注意的是需要需要下载的是线刷包，并且小米手机已经解开了bl锁，连接手机使用命令查看设备号，只有出来设备号才行，如果没有设备号就是没有把usb调试打开

adb devices

记住这个设备号，输入代码让手机进入fastboot模式，为了保险起见再查询一遍设备号，注意这里指令不同是因为手机进入了fastboot模式，输入代码

fastboot devices

随后使用“cd”指令进入线刷包目录，这里直接输入cd后把线刷包拖入终端

cd 线刷包存放地址

把刚刚记下的设备号加上指令写入回车等待刷入完成即可，指令如下：

sh ./flash_all.sh -s 设备号
